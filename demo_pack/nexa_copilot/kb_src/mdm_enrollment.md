# NexaCorp MDM Enrollment Guide

## Supported Platforms

- macOS 13+ (corporate laptops)
- iOS 16+ (BYOD with approval)
- Windows 11 (engineering workstations)

## Enrollment Steps (macOS)

1. Download the NexaCorp MDM profile from the self-service portal.
2. Open System Settings → Profiles and install the NexaCorp Device Management profile.
3. Authenticate with your NexaCorp SSO credentials.
4. Wait for compliance check (typically 5–10 minutes).

## Common Errors

- **MDM-101**: Profile expired — re-download from portal
- **MDM-204**: Device not in approved inventory — contact IT provisioning
