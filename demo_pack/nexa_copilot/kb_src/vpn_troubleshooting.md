# NexaCorp VPN Troubleshooting

## Common Issues After macOS Updates

After a major macOS update, employees may experience VPN disconnections. Recommended steps:

1. Restart the NexaVPN client from the menu bar.
2. Remove and re-add the VPN profile in System Settings → VPN.
3. Ensure the NexaCorp root certificate is trusted in Keychain Access.
4. Contact IT if error code VPN-403 persists after 15 minutes.

## Password Reset

VPN passwords sync with your NexaCorp Active Directory account. Reset via the self-service portal at https://sso.nexacorp.internal/reset.

## Support Escalation

If issues persist, open ticket INC-* via the IT portal or email helpdesk@nexacorp.internal.
